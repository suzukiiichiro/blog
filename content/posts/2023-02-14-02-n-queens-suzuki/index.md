## プログラムに置き換えてみます。

まず、NQueen.sh ファイルを作成して以下を書きます。


`function N-Queen01()`関数の中身はまだなにもありません。

``` bash
function N-Queen01(){
  local -i min="$1";
  local -i size="$2";

```
`function N-Queen01()`関数の冒頭で、渡された２つのパラメータを変数に格納しておきます。
この関数でしか使わないローカル関数なので、明示的に`local`としておきましょう。
`-i`は代入される値は数値であることを定義しています。
`$1` や `$2` といった変数はかならずダブルクォーテーションで囲む癖をつけておいてください。

## ソースコード
では、プログラムを見てみましょう。

``` bash:N-Queens.sh
#!/usr/bin/bash

#
declare -i COUNT=1;     # カウンター
#
: '
  function N-Queen1()
  <>１．ブルートフォース（力まかせ探索） N-Queen1()
  min : $1
  size : $2
';
function N-Queen01(){
  local -i min="$1";
  local -i size="$2";
  local -i i=0;         # 再帰に必要
  local -i j=0;         # 再帰に必要
  local sEcho="";       # 出力用変数
  for((i=0;i<size;i++)){
    pos[$min]="$i";
    if((min==size-1));then
      # echo で出力すると遅くなるので
      # 変数に入れておきます。
      sEcho="$((COUNT++)): "; 
      for((j=0;j<size;j++)){
        # echo で出力すると遅くなるので
        # 変数に入れておきます。
        sEcho="${sEcho}${pos[j]}";
      }
      # ここで一気に出力します。
      echo "$sEcho"     # flush出力
    else
      N-Queen01 "$((min+1))" "$size"; # 再帰
    fi
  }
}
#
function NQ(){
  echo "<>１．ブルートフォース（力まかせ探索） N-Queen1()";
  N-Queen01 0 5;
}
# 
NQ;
```

## ひとこと
　ページの冒頭では、左側の列が最も激しく動き、右へ右へと完成されていく挙動でした。
　プログラムソースでは、最も右側の列が激しく動き、桁上りで左左への完成されていきます。
　これは、プログラムの出力や、桁上りの特性上、逆になるのは致し方ないのです。
　頭を切り替えて考えてください。処理内容としては左右が反転するだけで同じことです。


今回は、縦列に一つだけのクイーンを配置するという一つのルールで処理しました。
次回は、横行に一つだけのクイーンを配置するというルールを追加して処理してみます。

これはだめで
```

        +-------------------+
```

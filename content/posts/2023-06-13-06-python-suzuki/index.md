## 割り算の値を切り捨てで扱いたいのですが？

切り捨て除算といいます。
除算の商の小数でもなく、余りでもなく、整数部分がほしいときはどうすればいいかということですよね。
そのような計算を行えるのが「切り捨て除算」であり、Pythonではそのための演算子が用意されています。
切り捨て除算を行うための演算子は、ダブルスラッシュ `//` です。

この場合も、下のような形で記述することで、aをbで割った商の整数部分を得ることができます。

```python
a // b
```
